# TypeScript Express Template

Helps to quickly setup TypeScript projects with an ExpressJS server.

## Dependencies

- NodeJS 18+
- Docker

## Usage

```sh
docker run --init -p 7000:7000 experimentalsoftware/fake-server
```

## Development

```sh
npm run dev
```
